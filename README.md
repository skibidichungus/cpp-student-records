# C++ Student Records – Phase 6

This project implements a basic student records system in C++.

## Features
- Read student data from files
- Store grades using a Student class
- Compute averages, min/max, and letter grades

## Files
- `Student.h / Student.cpp` – Student class implementation
- `main.cpp` – Program entry point
- `students.txt`, `grades.txt` – Input data

## Build
Open the `.sln` file in Visual Studio and build in Debug or Release mode.
